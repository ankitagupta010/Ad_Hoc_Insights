# Ad_Hoc_Insights

# Company: AtliQ Hardwares

AtliQ Hardwares is a fictional company that manufactures computer hardware in India and serves customers across the world. The company aims to use data more effectively to make faster and smarter business decisions.

# The Challenge

AtliQ Hardwares realized that their team needed to improve their ability to quickly generate insights using data. To do this, they planned to hire junior data analysts with strong technical and communication skills.
To test these abilities, Tony Sharma, the Director of Data Analytics, created a SQL-based challenge.

## My Role

As a candidate for this position, I had to:

Go through the ad-hoc-requests.pdf, which included 10 business-related data questions.

Write SQL queries to answer each request.

Build a presentation for top management to explain the insights clearly.

## My Process

## SQL Querying

Used MySQL to write and run queries to collect the required data.

## Data Visualization

Used Power BI to turn the data into easy-to-read visuals.

## Presentation Building

Created a clean and professional PowerPoint presentation to share the findings.

## Business Insights

Added meaningful recommendations to help the management team make better decisions.

## Result

This project allowed me to showcase my ability to work with data from start to finish—writing SQL queries, analyzing results, creating visualizations, and presenting insights in a simple and effective way.
It improved both my technical skills and my ability to clearly communicate data insights.

## What’s Included

ad-hoc-requests.pdf – The list of business questions to solve

SQL Queries – Folder with all SQL scripts used

Power BI Visuals – Folder containing Power BI files

Presentation – PowerPoint file with insights and recommendations

How to Use

SQL Queries

Open the SQL Queries folder and run the scripts in MySQL.

Power BI Visuals

Open the files in Power BI Desktop to view dashboards and charts.

Presentation

View the PowerPoint to see the complete summary of insights.
